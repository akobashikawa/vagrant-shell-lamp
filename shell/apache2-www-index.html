Remember

  • Change to folder where Vagrantfile is:
    • vagrant ssh to enter in guest
    • vagrant suspend to stop vagrant and save guest status
    • vagrant halt to stop vagrant and shutdown guest
    • vagrant up to start vagrant
    • vagrant destroy to stop vagrant and destroy all, if you wish start from scratch again
  • Web folder is shared:
    • In guest: /vagrant/www/
    • In host: ./www/
  • To change domain name http://host.dev to other you need to edit:
    • apache2's virtual host definition (/etc/apache2/sites-available/development)
    • guest's host file
    • host's host file
